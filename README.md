# スタジオしまづ事件簿 カチカチ時計と消えたチョコレート
バレンタインデーにチョコをもらった勇者が,誰かにチョコを盗まれてしまいその犯人を探すお話  

# どんなゲーム
・ゲームではないです  
・ストーリーを楽しむためのもの  
・Fungusなどを使わずにゲームイベントの作成をしました
